Source code and executables for Roberts et al. (2016), Correlation confidence limits for unevenly sampled data, Computers and Geosciences

This code produces 95% confidence intervals for the correlation between two data sets that may be unevenly and differently sampled using an accelerated bootstrap method

Manifest
fortran - directory containing Fortran source code
fortran/OpenACC - directory containg OpenACC version of the code
MATLAB - directory containing MATLAB source code
Windows - directory containing windows executable
Example - example data, in this case subsets of the MSA data, shorten to reduce execution time

readme.txt - this file
instructions.txt - instructions for the compilation and usage of the code, including the example data and expected output
